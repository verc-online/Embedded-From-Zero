# Embedded From Zero

> **Открытая книга по Embedded-разработке. От первого светодиода до реальных устройств.**

---

## Добро пожаловать!

**Embedded From Zero** — это открытая книга, которая показывает полный путь становления Embedded-разработчика.

Это не набор готовых библиотек и не сборник теории.

Это последовательное изучение Embedded Systems через создание собственных драйверов, чтение Datasheet, проектирование архитектуры и разработку реальных устройств.

Главная цель проекта — научиться понимать, **что происходит внутри микроконтроллера**, а не просто пользоваться готовыми функциями.

---

# Почему появился этот проект

Большинство материалов по Embedded можно разделить на две категории.

Первая — академические учебники, где подробно рассказывается теория, но почти нет реальных проектов.

Вторая — уроки, в которых достаточно вызвать готовую библиотеку, не понимая, как она работает.

Мне захотелось создать третью категорию.

Эта книга показывает весь путь обучения честно: с ошибками, вопросами, поиском решений и постепенным ростом от первых экспериментов до разработки полноценных устройств.

---

# Как читать эту книгу

Книга построена последовательно.

Каждая новая глава опирается на предыдущую.

Поэтому рекомендуется проходить материал именно в том порядке, в котором он расположен.

После каждой главы вас ждут:

* краткое повторение;
* практическое задание;
* вопросы для самопроверки;
* рекомендации по дальнейшему изучению.

# Основной проект

Во время изучения книги создаётся полноценное устройство — **автоматическая кормушка для домашних животных**.

В проекте реализованы:

* управление сервоприводом;
* работа с часами реального времени (RTC);
* сохранение настроек в EEPROM;
* LCD-интерфейс;
* меню пользователя;
* расписание кормлений;
* автоматическая выдача корма по времени.

Этот проект постепенно развивается вместе с каждой новой главой книги.
