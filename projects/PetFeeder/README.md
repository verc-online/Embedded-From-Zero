## 📖 О проекте

Pet Feeder — мой первый полноценный Embedded-проект, разработанный полностью с нуля.

Цель проекта — не просто собрать автоматическую кормушку, а пройти полный инженерный цикл разработки:

- проектирование архитектуры;
- написание драйверов;
- работа с Datasheet;
- разработка конечного автомата (FSM);
- тестирование в Proteus;
- перенос на реальное устройство;
- отладка аппаратной части;
- работа с EEPROM и RTC.

Проект является частью моего пути становления Embedded-разработчиком.

## ✨ Возможности

- отображение текущего времени
- установка времени
- хранение расписания кормления
- добавление времени кормления
- удаление времени кормления
- автоматическая сортировка расписания
- сохранение расписания в EEPROM
- автоматическое восстановление после перезапуска
- управление сервоприводом
- меню на LCD1602
- управление тремя кнопками

## ⚙ Используемое оборудование

| Компонент | Назначение |
|-----------|------------|
| ATmega8 | микроконтроллер |
| LCD1602 | интерфейс пользователя |
| DS3231 | часы реального времени |
| SG90 | сервопривод |

## 🚀 Сборка

Компилятор:

```
avr-gcc
```

Частота:

```c
#define F_CPU 1000000UL
```

Прошивка:

```bash
avrdude -c usbasp -p m8 -B 16kHz -U flash:w:PetFeeder.hex:i
```
